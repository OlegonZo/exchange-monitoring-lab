# Лаборатория мониторинга бирж

Безопасный публичный демопроект на Python 3.11+: приведение данных MEXC и
Hyperliquid к единому формату, оценка состояния рынка и объяснимые правила
выхода из тестовой short-позиции.

[English README](README.md) · [Исследовательский кейс](docs/CASE_STUDY.md)

> Проект не выставляет ордера, не подключается к приватным API и не содержит
> торговую стратегию. Все данные синтетические, режим работы — только paper.

## Что демонстрирует проект

- единая типизированная модель данных для двух разных бирж;
- нейтральный ключ сравнения `BASE/USD` для рынков USD, USDT и USDC;
- точные расчёты цен и процентов через `Decimal`;
- проверка рыночного режима по BTC и ширине движения рынка;
- детерминированный автомат состояний: hard stop, take profit, breakeven,
  trailing stop и ограничение времени;
- текст уведомления, готовый для транспортного слоя Telegram, но без отправки;
- unit-тесты и GitHub Actions для Python 3.11–3.13.

## Быстрый запуск

```bash
python -m pip install -e .
python -m exchange_monitoring examples/sample_market_data.json
python -m unittest discover -s tests -v
```

Сторонние библиотеки и API-ключи не нужны.

`MarketSnapshot.symbol` — аналитический ключ, а не тикер для выставления ордера.
Рынок MEXC с котировкой USDT и бессрочный рынок Hyperliquid с расчётами в USDC
приводятся к `BASE/USD` только для сравнения. Реальная интеграция должна отдельно
хранить и проверять нативный символ каждой биржи.

## Почему опубликована именно демоверсия

В рабочем исследовательском окружении есть приватные параметры, реальные логи,
адреса и секреты. Их публикация небезопасна. Здесь оставлена только архитектура,
которую можно проверить работодателю: нормализация данных, правила, обработка
ошибок, синтетический пример и тесты.

## Честно о результатах исследований

- Exit Engine, N=30: улучшение относительно baseline, но итог отрицательный —
  прибыльный edge не показан.
- Overheated Short, N=30: +183,26 paper USDT и 66,67% успешных наблюдений, но
  доверительный интервал пересекает ноль — edge не доказан.
- Market Regime Guard, N=30: +196,26 paper USDT и 70%, но интервал также
  пересекает ноль — нужна независимая проверка.
- Hyperliquid v1: N=0 из-за слишком строгих условий и ограничений данных.
- Hyperliquid v2: план зафиксирован заранее, исследование продолжается; результата
  пока нет.

Paper-результаты не равны реальной торговле. Проект не является финансовой
рекомендацией.
